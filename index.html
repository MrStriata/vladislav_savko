I have been programming for the last 3 years of my life (from 2017). Initially, there was a desire to become a game developer, and at this stage of life, two small games were written, as a kind of probe. One such game is an imitation of a slot machine where you need to get toys (Delphi), and the second was the famous Tetris (C# + Unity3D). Also in this period, a desktop application for service centers was created for accounting of customers and printing guarantees (C#). At the moment it is being converted to WEB.
Then my eyes fell on the development of web applications using ASP.NET Core MVC technology. Consequently, the diploma project was written using this technology - an online store for the SIGN STREETWEAR clothing brand, as well as a project for gyms - an admin panel. The last project was an online store for the CEREMONY clothing brand, where I was a front-end developer.
About me: sociable, I quickly learn the material, I can work in a team, I easily adapt.
